质数

  • 质数：在大于 1 的整数中，如果只包含 1 和本身这两个约数，就被成为质数，或者叫素数。
  • 算术基本定理：任一大于 1 的整数 \(n\)​ 能够唯一地表示成

\[ n ={p_1}^{\alpha _1}{p_1}^{\alpha _1}\cdots {p_s}^{\alpha _s}, \alpha _i > 0, i = 1,2,\cdots , s\left( * \right) \]

​ 分解式 \((*)\) 叫做 \(n\) 的标准分解式

注：有时为了应用方便，在分解式中插入若干素数的零次幂，而把 \(n\) 表成下面形式： \[ n ={p_1}^{\alpha _1}{p_1}^{\alpha _1}\cdots {p_k}^{\alpha _k}, \alpha _i\geqslant 0, i = 1,2,\cdots , k \]

判定质数——试除法

【定理】一个数的因数都是成对出现的（例如 12 的因数有 3 和 4，2 和 6）所以我们可以只枚举较小的那一个。假设较小的为 \(d\)，较大的为 \(n/d\)，则： \[ d\leqslant \frac{n}{d}\Longrightarrow d\leqslant \sqrt{n} \]

【时间复杂度】\(O(\sqrt{n})\)​

注意第 5 行：

  • \(i\) 必须从 \(2\) 开始，从 \(1\) 开始会死循环
  • 循环判断是 \(\le\) 而不是 \(<\)
  • 不要写成 \(i*i<= n\)，因为这样在 \(i\) 非常大时会爆掉 \(INT\)
  • 也不要写成 \(𝑖<=sqrt(𝑛)\)，否则每次判断循环条件是否成立都会调用 sqrt() ，时间复杂度较高 \(O(n^\alpha)\)

朴素筛法（不用）

【时间复杂度】\(O(n \log n)\)​ \[ \frac{n}{2}+\frac{n}{3}+\cdots +\frac{n}{n}= n\left( \frac{1}{2}+\frac{1}{3}+\cdots +\frac{1}{n} \right) \approx n\log n \]

【原理】任意 整数 \(x\) \((x>1)\)​ 的倍数 \(2x\)，\(3x\)，… 等都不是质数

  • 原理：任意 素数 \(x\) \((x>1)\) 的倍数 \(2x\)，\(3x\)，… 等都不是质数

线性筛法

  • 朴素筛法和 \(Eratosthenes\) 筛法存在 重复筛掉 的情况

例如 \(12\) 既会被 \(2\) 筛掉，又会被 \(3\) 筛掉：

  1. 在标记 \(2\) 的倍数时，\(12 = 2 ∗ 6\)
  2. 在标记 \(3\) 的倍数时，\(12 = 3 ∗ 4\)

如何避免某个数被重复筛掉的情况呢？很简单，我们只需让 \(12\) 只被它的 最小质数 \(2\) 筛掉即可

  • 原理：合数 \(x\) 只会被其 最小质因子 筛掉一次
  • 时间复杂度：\(O(n)\)​

第二层循环 st[primes[j] * i] = true; 只会对所有 合数 执行一次，所以时间复杂度为 \(O(n)\)

为什么不需要写 j < cnt ？

  1. primes[] 数组中存有 \(1\) ~ \(i\) 的所有质数

  2. 当 \(i\) 是合数时，肯定会在中间就 break 掉

  3. 当 \(i\)​ 是质数时, 那么 primes[cnt - 1] = i，因此当 j == cnt - 1 时，i % primes[j] == 0, break，依然不会越界

为什么循环条件是 primes[j] <= n / i

  1. 循环内筛掉的数是 primes[j] * i
  2. 我们要求的是 \(1\) ~ \(n\) 之间的所有素数
  3. 当 primes[j] > n / i，下面筛掉的数大于 \(n\)，不符合我们的要求

证明

  • 合数 x 的最小质因子只有一个

显然成立

  • 合数 x 只会 被其 最小质因子 筛掉 一次

  • 所有的合数一定会被筛掉

证明：假设 \(prime[k]\) 是合数 \(x\) 的最小质因子。当 \(i\) 枚举到 \(x/primes[k]\) 的时候，\(x\) 一定会被 \(prime[k]\) 筛掉，即 st[primes[k] * (x / primes[k])] = true

因此，所有合数一定 只 会被 最小质因子 筛掉一次，时间复杂度是线性的

约数个数

  • 约束个数 定理：任一大于 \(1\) 的整数 \(n\)​ 分解质因式：

\[ n ={p_1}^{\alpha _1}{p_1}^{\alpha _1}\cdots {p_k}^{\alpha _k}, \alpha _i\geqslant 0, i = 1,2,\cdots , k \]

​ 则 \(n\) 的正约数的个数为 $( _1+1 ) ( _1+1 ) ( _k+1 ) $

  • 证明：\(n\) 可以分解质因数：\(n=p_{1}^{a_1}\cdot p_{2}^{a_2}\cdot p_{3}^{a_3}\cdots p_{k}^{a_k}\)，由约束定义可知

​ \(p_{1}^{a_1}\) 的约数有 \(p_{1}^{0},p_{1}^{1},p_{1}^{2},\cdots ,p_{1}^{a_1}\)，共 \(a_1+1\) 个

​ \(p_{2}^{a_2}\) 的约数有 \(p_{2}^{0},p_{2}^{1},p_{2}^{2},\cdots ,p_{2}^{a_2}\)，共 \(a_2 +1\)​ 个

​ \(\dots\)

​ \(p_{k}^{a_k}\) 的约数有 \(p_{k}^{0},p_{k}^{1},p_{k}^{2},\cdots ,p_{k}^{a_k}\)，共 \(a_k+1\)​ 个

故故根据乘法原理， \(n\) 的正约数的个数为 $( _1+1 ) ( _1+1 ) ( _k+1 ) $

约数之和

【约束之和定理】任一正整数 \(n(n>=2)\) 分解质因式： \[ n ={p_1}^{\alpha _1}{p_1}^{\alpha _1}\cdots {p_k}^{\alpha _k},\alpha _i > 0, i = 1,2,\cdots , k \]

​ 则 \(n(n>=2)\) 的正约数之和为 \(\left( p_{1}^{0}+p_{1}^{1}+\cdots +p_{1}^{\alpha _1} \right) \left( p_{2}^{0}+p_{2}^{1}+\cdots +p_{2}^{\alpha _2} \right) \cdots \left( p_{k}^{0}+p_{k}^{1}+\cdots +p_{k}^{\alpha _k} \right)\)

【证明】

若 \(n\) 可以分解质因数：\(n=p_1^{a_1}\times p_2^{a_2}\times p_3^{a_3}\times\cdots\times p_k^{a_k}\)

可知 \(p_{1}^{\alpha_{1}}\) 的约数为 \(p_1^0,~p_1^1,~p_1^2,\cdots,~p_1^{\alpha_1}\)

\(\cdots \cdots\)

同理可知，\(p_k^{\alpha_k}\) 的约数为 \(p_k^0, p_k^1, p_k^2, \cdots, p_k^{\alpha_k}\)

实际上 \(n\) 的约数是在 \(p_1^{\alpha_1},~p_2^{\alpha_2},\cdots,~p_k^{\alpha_k}\) 每一个的约数中分别挑一个相乘得来

可知共有 \((\alpha_1+1)(\alpha_2+1)(\alpha_3+1)\cdots(\alpha_k+1)\) 种挑法，即约数的个数。

由乘法原理可知它们的和为：\(f(n)=(p_1^0+p_1^1+p_1^2+\cdots+p_1^{\alpha_1})\cdot(p_2^0+p_2^1+p_2^2+\cdots+p_2^{\alpha_2})\cdot\cdots\cdot(p_k^0+p_k^1+p_k^2+\cdots+p_k^{\alpha_k})\)

【时间复杂度】\(O(\sqrt{n})\)

【注意】一个数的 约数之和 一定会 大于这个数本身（因为这个数本身也是自己的约数）

欧拉函数

求欧拉函数 \(\varphi \left( n \right)\)

  • 定理：设正整数 \(n\ (n>=2)\) 的标准分解式为：

\[ n ={p_1}^{\alpha _1}{p_1}^{\alpha _1}\cdots {p_k}^{\alpha _k},\alpha _i > 0, i = 1,2,\cdots , k \]

​ 则欧拉函数 \[ \varphi \left( n \right) = n\left( 1-\frac{1}{p_1} \right) \left( 1-\frac{1}{p_2} \right) \cdots \left( 1-\frac{1}{p_k} \right) \]

  • 含义： \(0/1\) ~ \(n-1/n\) 中与 \(n\)​ 互素 的 个数
  • 性质：
    1. 若 \(p\) 为素数，则 \(\varphi \left( p \right) = p-1\)
    2. 若 \((n, m) =1\)，则 $( mn ) =( m ) ( n ) $
    3. \(\varphi \left( a^b \right) =a^{b-1}\cdot \varphi \left( a \right)\)，\(a>=2\)
  • 时间复杂度：\(O(\sqrt{n})\)​

对比：如果对 \(1\) ~ \(n\) 每个数使用 3.1 计算 \(\varphi(i)\)，时间复杂度为 \(O(n \sqrt{n})\)

拓展欧几里得算法

【模板题： AcWing 877. 扩展欧几里得算法 - AcWing】

【时间复杂度】\(O(log(a+b))\)

求齐次方程 \(ax+by = 0\) 的解

\[ ax + by = 0 \\ x =-\frac{by}{a} \]

设 \(d = (a, b)\)，则 \(a = a_0 * d, b = b_0 * d\) 且 \((a_0, b_0) = 1\) \[ x =-\frac{b_0y}{a_0} \] 因为 \(a_0\)、\(b_0\) 互质， 则 \(x\) 有整数解 => \(a_0 | y\) => \(y = k * a_0\) ，其中 \(k\) 为任意整数 => 通解为 \[ x = - k * \frac{b}{(a, b)},\quad y = k * \frac{a}{(a, b)}\quad k\in \mathbb{Z} \] 其中 \(k\) 为任意整数

求 \(ax+by = gcd(a,b)\) 的解

【贝祖等式】 对任意 正整数 \(a, b\) ，一定存在整数 \(x, y\) 使得 \[ ax+by =\left( a, b \right) \]

注意 \(x, y\) 也可能为负数

（1）求解上述等式的 特解 \(x', y'\) ——拓展欧几里得算法

  • 当 \(b = 0\) 时 \(\quad ax + by = a\) 故而 \(x' = 1, y' = 0\)

  • 当 \(b ≠ 0\) 时

因为
\[ gcd(b, a\%b) = gcd(a, b) \] 而 \[ bx'' + (a \% b)y'' = gcd(b, a\%b) \]

\[ bx'' + (a-\lfloor a/b \rfloor*b)y'' = gcd(b, a\%b) \]

\[ ay'' + b(x''-\lfloor a/b \rfloor*y'') = gcd(b, a\%b) = gcd(a, b) \]

故而

\[ x' = y'' , \quad y' = x'' - \lfloor a/b \rfloor*y'' \] 因此可以采取递归算法 先求出下一层的 \(x''\) 和 \(y''\) 再利用上述公式回代即可

如果中间运算过程中 \(x,y\) 爆掉 INT，则将其类型修改为 LL

（2）求解上述等式的 通解 \(x, y\)

通解 = 特解 + 齐次解，而齐次解即为方程 \(ax + by = 0\) 的解，故而通解为 \[ x = x’+k *\frac{b}{(a, b)},\quad y = y’-k*\frac{a}{(a, b)}\quad k\in \mathbb{Z} \] 若令 \(t=b/d\)， 则对于 \(x\) 的 最小非负整数解 为 \((x’\%t+t)\%t\)

求 \(ax + by = c\) 的解

【定理】对任意正整数 \(a, b\) ：存在 \(x,y\) 使得 \(ax + by = c\) 等价于 \((a,b)|c\)

证明

（1）正向：因为 \(a,b\) 是 \((a,b)\) 的倍数，则 \(ax + by=c\) 一定也是 \((a,b)\) 的倍数，所以 \((a,b)|c\)

（2）反向：由贝祖等式知，一定存在 \(x_0,y_0\) 使得 \(ax_0+by_0=\left( a,b \right)\)，因此两边同时乘以 \(\frac{c}{(a,b)}\)【反向证明已知条件 \((a,b)|c\) 】，得到 \(a\frac{x_0c}{\left( a,b \right)}+b\frac{y_0c}{\left( a,b \right)}=c\)，即存在特解 \(x=\frac{x_0c}{\left( a,b \right)},y=b\frac{y_0c}{\left( a,b \right)}\) 使得 \(ax + by = c\)

注意：该定理结合贝祖等式，说明由 \(x\) 和 \(y\) 的倍数之和凑出的最小正整数是 \((a,b)\)，所有凑出来的数都是 \((a,b)\) de

（1）求解上述等式的 特解 \(x', y'\) ？

  1. 使用 拓展欧几里得算法 ，求出 \(x_0\)、\(y_0\)，使得 \(ax_0+by_0=(a,b)\)

  2. 左右两侧同时乘以 \(\frac{c}{\left( a,b \right)}\)，得到 \[ a\frac{x_0c}{\left( a, b \right)}+b\frac{y_0c}{\left( a, b \right)}= c \]

  3. 因此特解

\[ x'=\frac{x_0c}{\left( a, b \right)},\quad y'= b\frac{y_0c}{\left( a, b \right)}\quad k\in \mathbb{Z} \]

（2）求解上述等式的 通解 \(x, y\)

通解 = 特解 + 齐次解，而齐次解即为方程 \(ax + by = 0\) 的解，故而通解为 \[ x = x’+k *\frac{b}{(a, b)},\quad y = y’-k*\frac{a}{(a, b)}\quad k\in \mathbb{Z} \] 若令 \(t=b/d\)， 则对于 \(x\) 的 最小非负整数解 为 \((x’\%t+t)\%t\)

（1）判断 \(a^{-1}\) 是否存在 \[ a^{-1}\text{存在}\Longleftrightarrow gcd\left( a, p \right) = 1\xLeftrightarrow{p\text{为素数}}a\not \equiv 0\,\, mod\,\, p \] （2）若 \(a^{-1}\) 存在，则 \[ a^{-1}= a^{p-2}\, \, \left( mod\,\, p \right)\quad s.t. a\not \equiv 0\,\, mod\,\, p \]

  • 时间复杂度：\(O(logp)\)

求解同余式

【定理】 一次同余式 \[ ax\equiv b\left( mod\,\, m \right) \text{，}a\not \equiv 0\left( mod\,\, m \right) \]

（1）有解 \(\Longleftrightarrow \left( a,m \right) |b\)​

（2）当同余式有解时，其解数为 \(d=(a,m)\)​​

（3）求特解 \(x\)，满足 \(ax≡b(mod \ m)\) <==> 求特解 \(x\)、\(y\) 满足 \(ax+my=b\)

  1. 使用 拓展欧几里得算法 ，求出 \(x_0\)、\(y_0\)，使得

\[ ax_0+my_0 =(a, m) \]

  1. 左右两侧同时乘以 \(\frac{b}{\left( a,m \right)}\)，得到

\[ a\frac{x_0b}{\left( a, m \right)}+m\frac{y_0b}{\left( a, m \right)}= b \]

  1. 因此特解

\[ x =\frac{x_0b}{\left( a, m \right)}\quad \% m \]

特别的 当 \(b = 1\) 且 \(a\) 与 \(m\) 互质时 则所求的 \(x\) 即为 \(a\) 的逆元

NIM 游戏

定理

​ 假设 \(n\)​ 堆石子，数目分别是 \(A_1,A_1,\cdots,A_n\)​

  • 如果 \(A_1\oplus A_2\oplus \cdots \oplus A_n\ne 0\)​​，则先手必胜
  • 否则先手必败

证明

  • 操作到最后时，每堆石子数都是 \(0\)，\(0 \oplus 0 \oplus … 0 = 0\)

  • 在操作过程中，如果 \(a_1 \oplus a_2 \oplus … \oplus a_n = x \neq 0\)。那么玩家必然可以通过拿走某一堆若干个石子将异或结果变为 0

    证明：不妨设 x 的二进制表示中最高一位 1 在第 k 位，那么在 \(a_1,a_2,…,a_n\) 中，必然有一个数 \(a_i\)，它的第 k 为时 1，且 \(a_i \oplus x < a_i\)，那么从第 \(i\) 堆石子中拿走 \((a_i - a_i \oplus x\))个石子，第 \(i\) 堆石子还剩 \(a_i - (a_i - a_i \oplus x) = a_i \oplus x\)，此时 \(a_1 \oplus a_2 \oplus … \oplus a_i \oplus x \oplus… \oplus a_n = x \oplus x = 0\)。

  • 在操作过程中，如果 \(a_1 \oplus a_2 \oplus … \oplus a_n = 0\)，那么无论玩家怎么拿，必然会导致最终异或结果不为 \(0\)

    反证法：假设玩家从第 \(i\) 堆石子拿走若干个，结果仍是 \(0\)。不妨设还剩下 \(a’\) 个，因为不能不拿，所以 \(0 \leq a’ < a_i\)，且 \(a_1 \oplus a_2 \oplus … \oplus a’ \oplus… \oplus a_n = 0\)。那么 \((a_1 \oplus a_2 \oplus … \oplus a_i \oplus… a_n) \oplus (a_1 \oplus a_2 \oplus … \oplus a’ \oplus… \oplus a_n) = a_i \oplus a’ = 0\)，则 \(a_i = a’\)，与假设 \(0 \leq a’ < a_i\) 矛盾

: tomato: 基于上述三个证明：

  1. 如果先手面对的局面是 \(a_1\oplus a_2\oplus \cdots \oplus a_n\ne 0\)，那么先手总可以通过拿走某一堆若干个石子，将局面变成 \(a_1\oplus a_2\oplus \cdots \oplus a_n=0\)。如此重复，最后一定是后手面临最终没有石子可拿的状态，先手必胜。
  2. 如果先手面对的局面是 \(a_1\oplus a_2\oplus \cdots \oplus a_n=0\)，那么无论先手怎么拿，都会将局面变成 \(a_1\oplus a_2\oplus \cdots \oplus a_n\ne 0\)，那么后手总可以通过拿走某一堆若干个石子，将局面变成 \(a_1\oplus a_2\oplus \cdots \oplus a_n=0\)。如此重复，后一定是先手面临最终没有石子可拿的状态，先手必败。

公平组合游戏 ICG

若一个游戏满足：

  1. 由两名玩家交替行动；
  2. 在游戏进程的任意时刻，可以执行的合法行动与轮到哪名玩家无关；
  3. 不能行动的玩家判负；

则称该游戏为一个 公平组合游戏。

​ NIM 博弈属于公平组合游戏，但城建的棋类游戏，比如围棋，就不是公平组合游戏。因为围棋交战双方分别只能落黑子和白子，胜负判定也比较复杂，不满足条件 2 和条件 3。

有向图游戏

​ 给定一个有向无环图，图中有一个唯一的起点，在起点上放有一枚棋子。两名玩家交替地把这枚棋子沿有向边进行移动，每次可以移动一步，无法移动者判负。该游戏被称为 有向图游戏。 ​ 任何一个公平组合游戏都可以转化为有向图游戏。具体方法是， 把每个 局面 看成图中的一个 节点，并且从 每个局面 向沿着合法行动能够到达的 下一个局面 连 有向边。

Mex 运算

​ 设 \(S\) 表示一个非负整数集合。定义 \(mex(S)\) 为求出 不属于 集合 \(S\) 的最小非负整数的运算，即： \[ mex(S) = min\{x\},\ x\ 属于自然数，且\ x\ 不属于\ S \]

SG 函数

​ 在有向图游戏中，对于每个节点 \(x\)，设从 \(x\) 出发共有 \(k\) 条有向边，分别到达节点 \(y_1, y_2, …, y_k\) ，定义 SG(x) 为 x 的后继节点 \(y_1, y_2, …, y_k\) 的 SG 函数值 构成的集合再执行 mex(S) 运算的结果，即： \[ SG(x) = mex({SG(y1), SG(y2), …, SG(yk)}) \] 特别地，整个有向图游戏 G 的 SG 函数值被定义为有向图游戏起点 s 的 SG 函数值，即： \[ SG(G) = SG(s) \]

有向图游戏的和

​ 设 G1，G2,····, Gm 是 m 个有向图游戏。定义有向图游戏 G，他的行动规则是任选某个有向图游戏 Gi，并在 Gi 上行动一步。G 被称为 有向图游戏 G1，G2，·····，Gm 的和

​ 有向图游戏的和的 SG 函数值等于它包含的各个子游戏 SG 函数的 异或和，即： \[ SG\left( G \right) = SG\left( G_1 \right) \oplus SG\left( G_2 \right) \oplus \cdots \oplus SG\left( G_m \right) \]

定理

  • 有向图游戏的某个局面 必胜，当且仅当该局面对应节点的 SG 函数值大于 0
  • 有向图游戏的某个局面 必败，当且仅当该局面对应节点的 SG 函数值等于 0

我们不再详细证明该定理。读者可以这样理解：

  • 在一个 没有出边 的节点上，棋子 不能 移动，它的 SG 值为 0，对应必败局面。
  • 若一个节点的某个后继节点 SG 值为 0，在 mex 运算后，该节点的 SG 值大于 0。这等价于，若一个局面的后继局面中存在必败局面，则当前局面为必胜局面。
  • 若一个节点的后继节点 SG 值均不为 0，在 mex 运算后，该节点的 SG 值为 0。这等价于，若一个局面的后继局面全部为必胜局面，则当前局面为必败局面。

对于若干个有向图游戏的和，其证明方法与 NIM 博弈类似。

方差与标准差

【均值】 \[ \bar{x}=\frac{1}{n}\sum_{i = 1}^n{x_i} \] 【方差】 \[ \sigma ^2 =\frac{1}{n}\sum_{i = 1}^n{(}x_i-\bar{x})^2 =\frac{1}{n}\sum_{i = 1}^n{x_{i}^{2}}-\bar{x}^2 \]

标准差 \[ \sigma =\sqrt{\frac{1}{n}\sum_{i = 1}^n{(}x_i-\bar{x})^2}=\sqrt{\frac{1}{n}\sum_{i = 1}^n{x_{i}^{2}}-\bar{x}^2} \]

推导 \[ \begin{aligned} &\frac{\sum_{i = 1}^n{(}x_i-\bar{x})^2}{n}\\ &=\frac{1}{n}*\sum_{i = 1}^n{(}x_{i}^{2}-2x_i\bar{x}+\bar{x}^2)\\ &=\frac{1}{n}(\sum_{i = 1}^n{x_{i}^{2}}-2\bar{x}*\sum_{i = 1}^n{x_i}+n\bar{x}^2)\\ &=\frac{1}{n}(\sum_{i = 1}^n{x_{i}^{2}}-2n\bar{x}^2+n\bar{x}^2)\\ &=\frac{1}{n}(\sum_{i = 1}^n{x_{i}^{2}}-n\bar{x}^2)\\ &=\frac{1}{n}\sum_{i = 1}^n{x_{i}^{2}}-\bar{x}^2\\ &=\frac{1}{n}\sum_{i = 1}^n{x_{i}^{2}}-\left( \frac{1}{n}\sum_{i = 1}^n{x_i} \right) ^2\\ \end{aligned} \]

注意：一定要会方差公式的 推导变换
