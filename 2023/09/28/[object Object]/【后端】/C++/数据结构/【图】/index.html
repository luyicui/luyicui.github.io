邻接表(Adjacency List）

  1. 邻接表是图的链式存储结构
  2. 只存储图中存在的边的信息，对不相邻的顶点则不保留信息
  3. 对图中每个顶点vi建立一个带头结点的单链表，称为顶点表
  4. 把与vi相邻接的顶点放在这个链表中，一个单链表对应邻接矩阵中的一行，称为边链表。

存储结构

分三部分： 1、图的结构定义 2、顶点的头结点结构 3、弧（边）的结点结构

2，采用邻接表创建无向图

【算法步骤】

1，输入总顶点数和总边数

2，建立顶点表

  • 依次输入点的信息存入顶点表中
  • 使每个表头结点的指针域初始化为NULL

3、创建邻接表

  • 依次输入每条边依附的两个顶点
  • 确定这两个顶点的序号i和j
  • 将此边结点分别插入vi和vj对应的两个边链表的头部

注意

  1. 顶点表顺序存储，以便可以随机访问任一顶点的边链表
  2. 邻接表表示不唯一

最小生成树

  • 生成树：包含==连通图==中所有顶点的极小连通子图

  • 最小生成树（Minimum Spanning Tree，简称MST）：带权连通图（网）的所有生成树中，各边权值之和最小的生成树称为最小生成树

  1. 只有连通图（无向）才存在最小生成树/生成树

  2. 有向图不存在生成树的概念

  3. 连通图的生成树具有这样的特征，即生成树中边的数量 = 顶点数 - 1。

prim算法(加点法)

算法概要

**设G = （ V , E )是连通网，T是 G 上最小生成树中边的集合。U为G上最小生成树顶点的集合**

（1）初始时，U={u0}（u0∈V）,T={ }

（2）找到满足min{ weight(ui,vi) |ui∈U，vi∈V-U}的边(ui,vi)，并将该边加入T，同时vi并入U；

（3）重复步骤（2），直到U=V时算法结束

数据结构

（1）图采用邻接矩阵来存储 （2）一维数组closedeg,记录从U到V-U具有最小代价的边。

closeedge[i]表示顶点==vi==到集合U的最短边，其中

  • closeedge[i].adjvex表示依附于这条最小代价边的另一个顶点
  • closeedge[i].lowcost表示最小边的权值
    • closeedge[i].lowcost = 0 :表示顶点 vi 已经在顶点集U中
    • closeedge[i].lowcost != 0 :表示顶点 vi 还在顶点集V-U中

closeedge[i];有三种功能

  1. 标记顶点 vi是否已经再顶点集U中
  2. 记录顶点 vi到集合U-V的最小边（包含另一个顶点adjvex和边的权值）

注意

Dijkstra算法不能处理带有负权值的边

  • 反例

按照Prim算法，d（4）为2，路径为1，3，4

但显然路径是1，2，3，4时，d（4）才是最小，为1

  • 原因：Dijkstra算法是按照路径长度递增的次序，逐步产生各个顶点到源点最短路径的贪心算法，而当存在带负权值的边时，当把一个节点选入集合S时，即意味着已经找到了从源点到这个点的最短路径，但若存在负权边，就与这个前提矛盾，可能会出现得出的距离加上负权后比已经得到S中的最短路径还短。（无法回溯）

Floyd算法允许有带负权值的边，但不允许图中有包含带负权值回路

  • 这句话的意思并不是说“只要回路中存在负权值Floyd就不可以解决”。而是“组成这个回路的所有的边的权值之和如果为负，就无法解决，否则还是可以解决的”

  • 举例说明

这个图中右侧的三个边组成了一个回路，其中一边为负，但是三边权值之和为：2+3+(-1)=4 >0，此时Folyd算法是可以解决的

而第二张图同样右侧的三个边组成了一个回路，其中一边为负，但是三边权值之和为：1+1+(-3)=-2<0，此时Folyd算法无法解决，因为每多走一遍回路总的路径就会 -2 如果求路径长度最小值只需要一直走下去，进入了死循环。

拓扑排序

定义

  1. 顶点活动网(AOV网，Activity On Vertex NetWork)：将==顶点表示活动，边表示活动之间的关系==的有向网称为顶点活动网；

  2. 拓扑序列：若有向图G/AOV网的顶点序列v1,v2,…,vn满足下列条件：若在AOV网/有向图G中存在从vi到vj的路径，则在该（线性）序列中的vi必定在vj之前，便称这个序列为一个拓扑序列。

  3. 求一个有向图拓扑序列的过程称为拓扑排序。

算法概要

拓扑排序可以判断一个有向图是否存在环路

  1. 若网中所有顶点都在其的拓扑排序序列中，则该AOV网中必不存在环。
  2. 如果有向图存在环，则不能进行拓扑排序；反之，如果对一个有向图不能进行拓扑排序，则必定存在环。
    • ==有向图存在环路< = = >不能进行拓扑排序< = = >不存在关键路径==
    • ==有向图不存在环路（即有向无环图）< = = >能进行拓扑排序< = = >存在关键路径==
